# Bootstrap 5 Exercises

This repository contains solutions for Bootstrap 5 exercises completed as part of Full Stack Development learning.

# Topics Covered

* Bootstrap Setup
* Bootstrap Grid System
* Containers, Rows and Columns
* Flexbox Utilities
* Typography
* Forms and Input Groups
* Buttons and Button Groups
* Navigation Bars
* Cards and Media Objects
* Spacing Utilities
* Colors and Backgrounds
* Display and Visibility Utilities
* Borders and Shadows
* Position Utilities
* Bootstrap Icons
* Modals and Accordions
* Bootstrap Installation using npm
* Bootstrap Customization using Sass


# Technologies Used

* HTML5
* Bootstrap 5

# Purpose

The purpose of this repository is to practice and understand Bootstrap 5 concepts through hands-on exercises and examples.
