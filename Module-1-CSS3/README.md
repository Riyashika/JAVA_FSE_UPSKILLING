# Community Event Portal – CSS3 Exercises
# Project Description

This project demonstrates the implementation of core CSS3 concepts using a single HTML file with an external stylesheet. It focuses on applying fundamental styling techniques, layout structures, and responsive design principles through practical exercises.

# Objectives Covered

Exercise 1 – CSS Types

Implementation of external CSS linking for styling the webpage.

Exercise 2 – CSS Syntax

Application of class-based styling for interactive elements such as buttons.

Exercise 3 – Selectors

Use of class and ID selectors to design structured content blocks.

Exercise 4 – Colors and Backgrounds

Application of gradient backgrounds and color styling for headings.

Exercise 5 – Typography

Improvement of text readability using alignment, spacing, and line-height.

Exercise 6 – Links and Lists

Styling of hyperlinks and removal of default list formatting.

Exercise 7 – Tables

Creation of a structured table with borders, padding, and alternating row colors.

Exercise 8 – Box Model

Demonstration of padding, borders, and visibility properties with form elements.

Exercise 9 – Multi-Column Layout

Implementation of multi-column text layout using CSS column properties.

Exercise 10 – Responsive Design

Use of Flexbox and media queries to ensure adaptability across different screen sizes.

# Technologies Used

HTML5
CSS3

# Responsive Features

Flexible navigation layout using Flexbox
Responsive image scaling for different screen sizes
Adaptive typography for smaller devices using media queries
