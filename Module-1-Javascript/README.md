# JavaScript Exercises – Local Community Event Portal

---

## Project Overview

This project demonstrates core and advanced **JavaScript concepts (1–14 exercises)** using a single interactive web application titled **Local Community Event Portal**.

It includes event management features such as registration, filtering, DOM manipulation, asynchronous data handling, form validation, and modern ES6+ features.

---

## Features Implemented

---

# Exercise 1 – JavaScript Basics & Setup

* Linked external JavaScript file using `<script src="main.js">`
* Used `console.log()` for debugging
* Displayed page load alert using `window.onload`

---

# Exercise 2 – Syntax, Data Types, and Operators

* Used `const` and `let` for variables
* Stored event details like name, date, and seats
* Used template literals for string formatting
* Managed seat count using increment/decrement operators

---

# Exercise 3 – Conditionals, Loops, and Error Handling

* Used `if-else` to validate event conditions
* Looping through events using `forEach()`
* Implemented error handling using `try-catch`

---

# Exercise 4 – Functions, Scope, Closures, Higher-Order Functions

* Created reusable functions for event operations
* Implemented closures to track registrations
* Used higher-order functions for filtering logic

---

# Exercise 5 – Objects and Prototypes

* Created Event objects using constructor function/class
* Added prototype method `checkAvailability()`
* Used `Object.entries()` for object inspection

---

# Exercise 6 – Arrays and Methods

* Managed events using arrays
* Used `.push()` to add events
* Used `.filter()` to filter by category
* Used `.map()` for formatted display output

---

# Exercise 7 – DOM Manipulation

* Accessed DOM using `querySelector()` and `getElementById()`
* Dynamically created event cards using `createElement()`
* Updated UI based on user interactions

---

# Exercise 8 – Event Handling

* Implemented `onclick`, `onchange`, and `keydown` events
* Added interactive search functionality
* Handled real-time user input

---

# Exercise 9 – Async JavaScript (Promises, Fetch API)

* Fetched data using `fetch()`
* Used `.then()` and `.catch()` / `async-await`
* Displayed loading and fetched results from API

---

# Exercise 10 – Modern JavaScript Features

* Used ES6 features like:

  * `let`, `const`
  * Arrow functions
  * Default parameters
  * Destructuring
  * Spread operator

---

# Exercise 11 – Working with Forms

* Captured form input using `form.elements`
* Prevented default submission using `event.preventDefault()`
* Implemented form validation

---

# Exercise 12 – AJAX & Fetch API (POST Request)

* Sent data using `fetch()` POST method
* Simulated backend response using JSON placeholder API
* Displayed success/failure messages

---

# Exercise 13 – Debugging and Testing

* Used `console.log()` for debugging
* Used `debugger` keyword for breakpoints
* Inspected variables using browser DevTools

---

# Exercise 14 – jQuery (Conceptual)

* Demonstrated basic jQuery usage (click, fadeIn, fadeOut)
* Understood difference between jQuery and modern frameworks
* Mentioned transition towards frameworks like React/Vue

---

## Technologies Used

* HTML5
* JavaScript (ES6+)
