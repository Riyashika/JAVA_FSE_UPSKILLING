# Core Java Exercises

## Overview

This repository contains solutions to a collection of Core Java programming exercises covering fundamental and advanced Java concepts.

## Topics Covered

- Java Basics
- Operators and Control Statements
- Methods and Recursion
- Arrays and Strings
- Object-Oriented Programming
- Inheritance and Interfaces
- Exception Handling
- File Handling
- Collections Framework
- Multithreading
- Lambda Expressions
- Stream API
- JDBC
- Java Modules
- Socket Programming
- Reflection API
- Virtual Threads
- Executor Service and Callable

## Exercises

The repository includes 41 exercises ranging from basic programs such as Hello World and Simple Calculator to advanced topics such as JDBC, Networking, Reflection, Virtual Threads, and Concurrency.
