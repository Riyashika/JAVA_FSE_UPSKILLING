# Local Community Event Portal – HTML5 Exercises

## Project Overview

This project is developed as part of HTML5 practice exercises focusing on building a **Local Community Event Portal**.
It demonstrates the implementation of semantic HTML, navigation, forms, multimedia, event handling, storage, and geolocation features.

All **10 exercises** are implemented in a **single HTML file** using internal CSS and JavaScript.

## Features Implemented

# Exercise 1 – HTML5 Base Template

* Created HTML5 document structure using `<!DOCTYPE html>`
* Added meta tags and language attribute
* Used comments for section organization

# Exercise 2 – Navigation and Linking

* Implemented navigation using `<nav>`
* Used anchor links for internal section navigation
* Added external link with `target="_blank"`

# Exercise 3 – Welcome Message with Styling

* Used `id` for unique banner styling
* Applied internal CSS for background color
* Used inline styles for highlighted text
* Demonstrated use of `.class` selectors
  
# Exercise 4 – Image Gallery

* Created table-based image gallery
* Used `<img>` with alt and title attributes
* Styled images using CSS classes
* Added event captions

# Exercise 5 – Event Registration Form

* Implemented form with input types: text, email, date, select, textarea
* Used validation attributes (`required`, `autofocus`)
* Displayed output using `<output>` element
* Styled form using CSS

# Exercise 6 – Event Feedback with Events Handling

* Used `onblur` for validation
* Used `onchange` to display event fee
* Used `onclick`, `ondblclick`, and keyboard events
* Displayed real-time character count
  
# Exercise 7 – Video Invite with Media Events

* Embedded video using `<video>` tag
* Used `oncanplay` event for readiness message
* Handled media interaction events

# Exercise 8 – Saving User Preferences

* Stored event type using `localStorage`
* Retrieved stored data on page reload
* Added clear function for `localStorage` and `sessionStorage`

# Exercise 9 – Geolocation Feature

* Used `getCurrentPosition()` API
* Displayed latitude and longitude
* Handled permission and timeout errors
* Enabled high accuracy mode

# Exercise 10 – Debugging with DevTools

* Used `console.log` for debugging
* Structured code for browser inspection
* Demonstrated runtime monitoring using DevTools

## Technologies Used

* HTML5
* CSS3 (Internal Styling)
* JavaScript (Client-side scripting)

## Conclusion

This project demonstrates foundational web development skills using HTML5, CSS3, and JavaScript to build an interactive community event portal.
