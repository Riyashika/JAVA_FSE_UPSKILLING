# ANSI SQL Using MySQL Exercises

## Overview

This repository contains solutions to a set of SQL exercises designed to demonstrate practical database querying and data analysis skills using MySQL.

The exercises cover various SQL concepts including joins, aggregations, subqueries, date functions, window functions, and reporting queries.

## Database Schema

The solutions are based on the following tables:

- Users
- Events
- Sessions
- Registrations
- Feedback
- Resources
